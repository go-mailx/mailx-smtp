# mailx-smtp

[![Go Reference](https://pkg.go.dev/badge/github.com/go-mailx/mailx-smtp.svg)](https://pkg.go.dev/github.com/go-mailx/mailx-smtp)

`github.com/go-mailx/mailx-smtp` is an SMTP adapter for [`github.com/go-mailx/mailx`](../mailx). It implements the `mailx.MailerAdapter` interface using [go-mail](https://github.com/wneessen/go-mail).

## Install

```sh
go get github.com/go-mailx/mailx-smtp
```

## Usage

See the [smtp example](https://github.com/go-mailx/mailx/tree/main/examples/smtp) in the main repo.

## Config

See [`pkg.go.dev/github.com/go-mailx/mailx-smtp`](https://pkg.go.dev/github.com/go-mailx/mailx-smtp) for all configuration options.
